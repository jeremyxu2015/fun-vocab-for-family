# 词趣家庭

一个面向家庭使用场景的背单词Web应用，让背单词成为亲子时光。

## 主要功能

### 家长功能
- 创建家庭，获取邀请码
- 管理孩子账号（查看学习进度、重置密码）
- 单词管理（手动添加、批量导入、PDF智能提取）
- 布置学习任务

### 孩子功能
- 学习新词（SM-2间隔重复算法）
- 复习单词
- 强化训练（错题攻坚、难词训练、拼写练习）
- 贪吃蛇背单词游戏
- 闯关游戏模式
- 联机对战

### PDF智能提取
- 上传PDF文件，使用Kimi AI智能识别英语单词
- 自动提取单词、音标、释义、例句
- 支持非结构化PDF（如课本扫描件、词汇表）

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 初始化数据库

```bash
python manage.py migrate
```

### 3. 运行服务器

```bash
python manage.py runserver
```

访问 http://127.0.0.1:8000 即可使用。

## 使用流程

### 家长注册
1. 访问首页，点击"家长注册"
2. 填写用户名、密码、家庭名称
3. 注册成功后获得家庭邀请码

### 添加孩子
1. 将邀请码分享给孩子
2. 孩子访问首页，点击"孩子注册"
3. 填写邀请码、用户名、密码、昵称等信息
4. 注册成功后自动加入家庭

### 导入单词
1. 家长登录后进入"PDF导入单词"
2. 上传包含英语单词的PDF文件
3. 系统自动识别并提取单词
4. 也可使用"单词管理"手动添加或批量导入

### 孩子学习
1. 孩子登录后可进行学习、复习、训练
2. 游戏模式让背单词更有趣
3. 家长可在控制台查看学习进度

## 技术栈

- 后端：Django 4.2
- 数据库：SQLite（开发）/ MySQL（生产）
- AI：Kimi API（月之暗面）
- PDF处理：PyMuPDF
